# Tooele County Jurisdictions Reference

## Overview
Tooele County has three primary land-use authorities relevant to development analysis:

| Jurisdiction | Type | Incorporated | ArcGIS Zoning Layer |
|---|---|---|---|
| Tooele County | County | N/A (unincorporated land) | 4 |
| Grantsville City | City | 1867 | 7 |
| Erda City | City | 2021 (very new) | 1 |
| Lake Point | Special Service District | — | 2 |
| Tooele City | City | — | 3 |

**Note:** Erda City incorporated in 2021. Many GIS datasets and assessor records still reference "Tooele County" for parcels that are now within Erda city limits. Always cross-check with the Municipalities spatial layer.

## Tooele County (Unincorporated) Zoning

| Code | Name | Notes |
|------|------|-------|
| A-1 | Agricultural | 5-acre min lots; farming, single-family |
| A-2 | Agricultural | Larger minimum; grazing emphasis |
| RR-1 | Rural Residential | 1–5 acre lots |
| R-1 | Single Family Residential | Standard suburban |
| R-2 | Medium Density Residential | Smaller lots / duplexes |
| MU | Mixed Use | Commercial + residential |
| C-1 | Neighborhood Commercial | Small retail |
| C-2 | General Commercial | Large retail / auto-oriented |
| M-1 | Light Industrial | |
| M-2 | Heavy Industrial | |

**Property Code Glossary (assessor codes)**
- `G` prefix = Greenbelt (agricultural tax deferral under Utah Code 59-2-503)
- `Z` = Class 2 agricultural
- `4` = Dry farmland

## Erda City Zoning
Erda adopted its own zoning ordinance post-incorporation. As of 2024 the code is still in development — many areas default to prior Tooele County zoning.

Planning Commission: Contact Erda City Hall for current schedule
Website: https://erdacity.com
