## Annexation policy
Grantsville requires that newly-annexed parcels be contiguous to existing
city limits. Annexation typically takes 6-9 months including the
required state lieutenant governor certification.

## Future Land Use Map
The General Plan's Future Land Use Map can show a different designation
than the current zoning — that's the "master-planned zone" the user is
asking about. A rezone application that aligns with the FLUM is much
more likely to succeed than one that conflicts with it.

## Where to verify
- Live zoning code: https://grantsvilleut.gov/ → Municipal Code → Title 17
- Future Land Use Map: General Plan PDF on city website
- Zoning map: usually a single PDF posted by the city; ask staff if not online
